# turbo_gpt_py_script

Chat real GPT in Python with `gpt-3.5-turbo`

## Install

### Dependences

Python3.x, pip

### Clone

`git clone https://github.com/Ph0enixxx/turbo_gpt_py_script && cd turbo_gpt_py_script`

### Install

`pip install -r requirements.txt`

## Usage

`python3 main.py <your-openAI-key>`
